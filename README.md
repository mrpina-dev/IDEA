## For Students

Assignments open directly in your browser. No download required.

1. Click the assignment link posted in Google Classroom
2. Complete all sections in your browser (progress saves automatically)
3. Use **Print / Export PDF** to generate your submission file
4. Upload the PDF to Google Classroom

\---

## For Instructor Use

To add a new assignment, upload the HTML file to this repository. It will be live at:

`https://mrpina-dev.github.io/IDEA/filename.html`

Naming convention: `idea113-blade-01.html`, `idea403-senior-progress.html`, etc.
