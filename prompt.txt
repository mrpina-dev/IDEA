Edit coins/index.html. Color code leaderboard cards by student section.

In the renderLeaderboard() function, add this section color map in JavaScript:

const SECTION_COLORS = {
  'IDEA-113':   '#00FF41',
  'IDEA-208-1': '#C8FF00',
  'IDEA-208-2': '#00F0FF',
  'IDEA-304.1': '#FF8C00',
  'IDEA-403':   '#FF88CC',
};

For each card:
1. Look up the student's section in SECTION_COLORS. Default to #4A7A52 if not found.
2. Apply a 3px solid left border to the card using that color at 60% opacity.
3. Apply a subtle top gradient inside the card using that color at 4% opacity (like the existing ::before pseudo but using the section color instead of gold).
4. Color the .coin-section text using that color at 80% opacity instead of var(--dim).
5. On card hover, bring the left border to 100% opacity.

Do not change card background colors. The existing red tint on negative cards should be preserved - just also apply the left border color on those cards.
